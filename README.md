# Pistons

HUD Técnico: Elemento de interface visual 2D com ancoragem absoluta na tela (matriz de coordenadas fixa). É renderizado em camada superior (Canvas Screen-Space) de forma persistente. Fornece feedback em tempo real de variáveis dinâmicas (HP, munição e progresso) sem obstruir o campo de visão central ou exigir pausa no fluxo de jogo.Mostrar código,

Este repositório contém um utilitário de atualização automática externa desenvolvido especificamente para o ecossistema Ubuntu (e derivados baseados em Debian). O software opera de forma totalmente isolada em segundo plano, evitando conflitos de permissão de escrita e travamento de arquivos do jogo principal durante o processo de hot-swap
